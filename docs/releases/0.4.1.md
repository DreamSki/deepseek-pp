## DeepSeek++ 0.4.1

0.4.1 是基于 0.4.0 的体验增强版本，重点引入 DeepSeek 页面悬浮宠物，并保持 README / release notes / 版本号与正式发布资产一致。

### 主要变化

- 新增「DeepSeek 小鲸鱼」悬浮宠物，可在 DeepSeek 页面展示思考、输出、工具执行、成功、失败和休眠等状态反馈。
- 设置页新增宠物开关、左下/右下固定位置、拖动自定义位置、尺寸、透明度和动态漂浮控制。
- 宠物开关、位置和外观配置保存在浏览器本地，刷新页面后继续生效。
- 将宠物图片资源纳入扩展可访问资源，确保 DeepSeek 页面中的悬浮宠物可以正常渲染。
- README 新增悬浮宠物功能介绍和 0.4.1 变更回顾，并继续保留 0.4.0 / 0.3.0 / 0.2.0 的折叠历史。
